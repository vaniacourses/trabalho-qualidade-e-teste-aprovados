# 💰 WinxBank - Sistema Bancário

Projeto desenvolvido pelo grupo Aprovados para a disciplina de **Qualidade e Teste de Software**.

O sistema escolhido dentre as aplicações listadas (https://github.com/orgs/repo-software-testing-courses/repositories) simula operações bancárias básicas. O objetivo é concluir os requisitos para a entrega seguindo boas práticas de teste aprendidas em sala.


# 🛠️ Ferramentas necessárias

- Java 17
- Maven
- JUnit 5
- Mockito
- Git / GitHub

## ✅ Boas Práticas

Para manter a qualidade e organização do projeto, siga as boas práticas abaixo:

- Nunca realizar alterações diretamente na branch `main`.

- Comentar qualquer commit.

- Manter a estrutura padrão.

- Garantir que arquivos desnecessários não sejam versionados.
